## 🖥️ 프로젝트 소개
스몰 럭셔리, 여행 수요에 호텔 인기가 급증함에 따라 모두가 찾고 싶어하는 호텔 프로젝트 기획 </br>
도담 : </br>
'어린아이가 탈 없이 잘 놀며 자라는 모양'을 뜻하는 순 우리말로 호텔을 찾는 고객 모두 머무르기를 바라는 마음으로 작명
